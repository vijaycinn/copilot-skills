## Notes on ACR Validation

Before publishing any ACR figure in a leadership update:

1. Pull actuals from `PerformanceSummary_AccountReport_*.csv` — this is ground truth
2. MSX "Consumed Recurring" on the opportunity = total lifetime milestone estimate — **not monthly run rate**
3. MSX milestone "Est. Monthly Use" = forward projection for when the milestone is realized — **not current consumption**
4. Always label March data as "partial month" with an asterisk

See `references/account-spotlight-format.md` for the correct format when writing Section 7 spotlights.
